# Resume Screening System

An AI-powered resume screening web application built using NLP techniques.

## 🔧 Tech Stack
- Python, Django
- NLP (TF-IDF, Cosine Similarity)
- MySQL
- HTML, CSS

## 💡 Features
- Upload resumes in PDF format
- Automatically ranks resumes based on job description
- Uses TF-IDF and Cosine Similarity for matching
